This mod compatible for version 1.21 of minecraft adds three unique items to the game, two of which are based on the necromancy theme -

The Sacrificial Rapier - This rapier can be used as a normal weapon but it has the speciality of making explosions summon on mobs after some hits which deals a lot of damage and it is also accompanied with great particle effects.

Necromancy Book - This cursed book contains three spells that can be used by the player for different purposes. These spells can be toggled by shift+right click and applied through simple right click on mobs.

The Void Attraction Spell - It can be used on one mob which will then attract every other mob within the chunk to the mob affected and will cause a huge explosion, killing the target as well as other mobs which could be useful in hordes.

The Suicide Curse - This curse makes the mob jump into lava voluntarily if the mob is near a lava source. It can be used to eliminate targets without much effort. If the mob is not near a lava, it chucks the mob away.

Telekinetic Grasp - This ability makes the player gain control over the movement of mobs. Through the necromancy book, players can use this spell to lock the mobs onto their crosshair and launch them using right click.

Grappling Hook - This grappling hook makes the movement a lot easier in the game. Players can use it by right clicking to deploy the hook and right clicking again to pull themselves in the direction of the hook. It implements momentum as well as trajectories to allow smooth movement for players.
